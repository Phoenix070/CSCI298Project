Help/Features

  • Databases Page

    • This page lists out all the databases in the mongoDB system. You can add/delete any database from the list.
    Clicking on the database name will lead to the "Collections Page" which displays all the collections in that database.


  • Collections Page

    • This page lists out all the collections in the selected database. You can add/delete/rename any collection from the list.
    Clicking on the collection name will lead to the "Collection Content Page" which displays all the documents in that collection.


  • Collection Content Page

    • This page lists out all the documents in the selected collection. You can do the following actions on any document from the list:
        

    • Add/Edit Attribute

      This allows you to add a new attribute and value OR edit existing attribute's value. If attribute exists then value can be changed, if not then attribute is added and value is set.

    • Delete Attribute

      This allows you to delete an attribute from the document. Only that document will be affected.

    • Delete Document

      This allows you to delete the whole document. It will ask for a confirmation before permanently deleting it.

    • Add New Document

      This allows you to add a new document to the collection. It will lead you to a new page where you can add attributes and values.
      After pressing done on "Add Document" page, it will add the document to the list.


  • Add Document Page

    • This page allows you to add attributes and values to the new document created in the collection.
    Enter attribute name and value in the textboxes and select "Single Value/List of Value".
    Then press the Add button and the key-value will be added to the document.
    Enter a comma-separated list in the value textbox if adding a list. Click Done button after adding all key-value pairs.


  • Tutorials Page

    • This page lists out all the tutorials provided. You can click on any desired tutorial and start learning.
    Try the commands in the mongo shell to get a full learning experience.


  • SQL to MongoDB Conversion Page

    • This page lists out all the basic SQL statements and the MongoDB translated statements.
    It also gives a basic idea of differences and similarities between the two.


  • Navigation

    • Buttons are available on each page to navigate freely to next/previous page.
    The top navbar and side-bar is placed to jump to necessary links.
