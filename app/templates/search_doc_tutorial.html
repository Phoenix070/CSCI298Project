Search Queries


The command for searching documents in MongoDB is: db.collection.find()
This command finds the document in the collection.


Search operation can be carried out as follows :

> db.collection_name.find( {Search condition} )

In this command, collection_name is the name of the collection search operation is performed on.
The document/documents are searched by using the search parameters and the document is retrieved.

Example :
> db.users.find() or > db.users.find( { } )
This command will retrieve all documents in the collection.

Example :
> db.users.find(
   { name: "Sherlock" }
)
The document is searched in the collection where name="Sherlock".
If the search matches multiple documents, then all those documents are retrieved.


The following functions can be used along with the search operation. These functions can be used individually or together depending on the requirements.
","
Logical AND Condition

Ex: > db.users.find(
   { name: "Sherlock", age: 25 }
) 		This is used for an AND conjunction of two or more search parameters.
Both conditions need to be satisfied to get the document.
$or
Logical OR Condition

Ex: > db.users.find(
   { $or: [ { name: "Sherlock" }, { age: 25 } ] }
)		 This is used for an OR conjunction of two or more search parameters.
Either condition needs to be satisfied to get the document.
$in

Ex: > db.users.find(
   { appliances: { $in: [ "Tablet", "Laptop" ] } }
)		 This is used to find using multiple values that match of the same attribute in a collection.
If attribute value is in any of the value list, then the document will be retrieved.
$lt, $lte, $gt, $gte

Ex: > db.users.find(
   { age: { $lte : 25 } }, salary: { $gt : 5000 } }
)		 This is used in the search parameters to check if attribute value is less than, less than equal, greater than or greater than equal to the number specified.
If <,<=,> or >= condition is satisfied then the document is retrieved.
count()

Ex: > db.users.find().count()
OR
Ex: > db.users.count()

Ex: > db.users.find( { age : 25 } ).count()		 This is used to get the count of the all documents in the collection.
If a condition is specified then it counts the number of retrieved documents.
limit()

Ex: > db.users.find().limit( 10 )

Ex: > db.users.find( { age : 25 } ).limit( 1 )		 This is used to limit the number of documents retrieved from the collection.
If a condition is specified then it retrieves those documents and then displays only the number of documents specified.
