MongoDB Overview

MongoDB derived from "humongous" is a document-oriented, NoSQL database. It is a non-relational database with JSON-like documents and dynamic schemas.
Internally it is stored in BSON(Binary JSON) format. It is an open-source software with highly varied applications, highly scalable and faster than traditional databases in many aspects.

MongoDB stores data in the form of documents, which are JSON-like field and value pairs.
An example of a document with field-value pairs:

      {
          id : "az2345",
          name : "John Doe",
          gender : "Male",
          age : 35,
          phone : ["232.1234", "232.1235"]
      }

A Collection in MongoDB stores all documents that are related and share a set of common attributes. Collections are just like tables expect they store non-relational data.

Collection :
{
   id : "az2345",
   name : "John Doe",
   gender : "Male",

Using Mongo shell for operating the database :

>> Open a terminal and type "mongo" to start the mongo shell. By default you get connected to database "test".
     OR
>> Type "mongo database_name" to start mongo shell and connect to the database name provided.

Use the following commands to start interacting with the database.

> use database_name
Ex: > use project		 This will connect to the database name provided if already exists.
If it doesn't exist, then database will be created and connected.
This command can also be used to switch from current database to the database name provided.
> db Check the currently selected database.
> show dbs Lists all the databases in the system.
> show collections Lists all the collections in the current database selected.
> db.createCollection(name)
Ex: db.createCollection("blog")		 To create a collection use this command. In this command, name is the collection name to be created.
> db.collection_name.command()
Ex: db.project.find() OR db.project.insert({id: 1})		 To use any CRUD operations on the collection, give the collection_name and the command to execute.
Commands can be insertion, deletion, modification or search operations.
> db.dropDatabase() To drop the selected database use this command. If no database currently selected, it will drop the "test" database.
> db.collection_name.drop()
Ex: db.blog.drop()		 To drop a collection use this command.
