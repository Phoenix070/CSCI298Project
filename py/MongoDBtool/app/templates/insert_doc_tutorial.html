Inserting Documents


The command for inserting documents in MongoDB is: db.collection.insert()
This command insert a new document in the collection with key-value pairs provided.


Insert operation can be carried out as follows :

> db.collection_name.insert( { key1: value1, key2: value2, ...} )

In this command, collection_name is the name of the collection insert operation is performed on.
The attributes of the document are the keys, and the values are the attribute values.

Example :
> db.users.insert(
      {
       id: "ab1234",
       name: "John Doe",
       gender: "Male"
       phone: ["232.1234", "232.1235"],
       }
  )

If the insert is successful, it will print this message :
> WriteResult({ "nInserted" : 1 })
The number in the message indicates number of inserts done. If number is 0 the insert was unsuccessful.

Another way to check if the insert was done by using the command :
> db.users.find()
This command will display the document inserted and any other previous documents inserted.

During insert operation an "_id" field is assigned to the document. An _id field can be provided in the insert to give it user assigned value.
Example :
> db.users.insert( { _id: 10, name: "Sherlock", salary: 2000 } )
This will overwrite the default value of the _id field.


To insert multiple documents in a single insert provide all the documents as a list to the insert function.

> db.users.insert( [
      {
       id: "ab1234",
       name: "John Doe",
       gender: "Male"
       phone: ["232.1234", "232.1235"],
       },
      {
       id: "az1245",
       name: "Jane Doe",
       gender: "Female"
       phone: ["232.1241", "232.1242"],
       },
      {
       id: "an2131",
       name: "Sherlock",
       gender: "Male"
       salary: 2000,
       },
   ] )
