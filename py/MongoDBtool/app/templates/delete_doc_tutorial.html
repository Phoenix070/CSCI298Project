Deleting Documents


The command for deleting documents in MongoDB is: db.collection.remove()
This command deletes the document in the collection.


Delete operation can be carried out as follows :

> db.collection_name.remove( { } )

In this command, collection_name is the name of the collection delete operation is performed on.
With this command all documents in the collection will be deleted.

Example :
> db.users.remove(
   { name: "Sherlock" }
)
The document is searched in the collection where name="Sherlock" and the retrieved document is deleted from the collection.

If multiple documents have to be deleted, then use multi=true as one of the arguments to the function.
Example :

> db.users.remove(
   { status: "A" },
   { multi: true }
)
This will find all the documents that satisfy the requirement and delete those documents.
